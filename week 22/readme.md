5. Есть следующий код:
    
    ```jsx
    let user = undefined;
    alert(`Привет, ${user.name}`);
    ```
    
    Как сделать так, чтобы при обращении к нему выводилось внятное сообщение об ошибке "Имя пользователя не заполнено"?
    

Ответ:
```js
let user = undefined;
try{
    alert(`Привет, ${user.name}`);
} catch(err){
    alert("Имя пользователя не заполнено");
}
```
